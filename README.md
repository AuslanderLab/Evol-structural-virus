# Evolutionary Virology in the Lens of Structural Biology

Code and scripts for all the analyses of manuscript J.Malnak *et al.* **Uncovering acquisition events and human-specific folds with pairwise comparisons of the predicted viral proteome.** (In preparation). 

Repo structure
-----------
+ `/intermediate_files` files used in this analysis
+ `/src` code for the major analysis used in this study
+ `/results` directory where results are deposited if `src` is rerun.



Source Data
-----------
Data used in this study is linked in our Zenodo, too big to be published on this github repo.
All Zenodo files can be put into the `intermediate_files` dir for downstream analysis
Files from the original database of protein fold clusters is available from the study J.Nomburg *et al.* **Birth of new protein folds and functions in the virome.** *Nature*. (2024) https://zenodo.org/records/10291581
Following files were used in the intermediate files directory: 
+ `merged_clusters.counts.tsv`
+ `merged_clusters.tax.tsv`
+ `ncbi_viruses_genome_lengths.csv`

Running the Analysis
---------------------
### Computing the distances between genomes based on structural and amino acid 
Files used in this analysis can be found under 
+ `/intermediate_files/nomburg_files/merged_clusters.counts.tsv`
+ `/intermediate_files/viral_prots_cluster.tsv` - protein clusters from mmseqs
+ `/intermediate_files/nomburg_prot_modified_species.tsv`- linking the protein clusters to the virus of origin
+ `/intermediate_files/header_both.csv`- the species represented both by structure and amino acid, these will be used to generate dataframes of matching species

In the Jupyter notebook file `make_df_taxonomic_counts.ipynb`, run main function to generate all the count dataframe files.

We also computed the jaccard index for all pairs of viruses
The following files were used in this analysis
+ `/intermediate_files/structure_df.csv` - binary matrix indicating of structural clusters in each virus, generated by the script above.
+ `/intermediate_files/amino_acid_count_df.csv`- binary matrix of presence of sequence clusters in each virus, generated by the script above.

In the Jupyter notebook file `metrics.ipynb` run main to generate the dataframe of pairwise distances between viruses.

### Running KNN analysis for Taxonomic Classification
Taxonomic classification is parsed `merged_clusters.counts.tsv` and offered as supplementary files.
Taxonomic labels provided below:
+ `/intermediate_files/phyla_species.tsv`
+ `/intermediate_files/class_species.tsv`
+ `/intermediate_files/order_species.tsv`
+ `/intermediate_files/family_species.tsv`
+ `/intermediate_files/genus_species.tsv`
  
To do classification as done in the manuscript run
```
# family classification for amino acids and structure-based features
./src/run_knn.sh ./intermediate_files/amino_acid_knn_df.csv ./intermediate_files/family_species.tsv ./results/knn_result_k_11_amino_acids.csv
./src/run_knn.sh ./intermediate_files/structure_knn_df.csv  ./intermediate_files/family_species.tsv ./results/knn_result_k_11_structure.csv
# perform classification for structure-based features at other taxonomic levels
./src/run_knn.sh ./intermediate_files/structure_knn_df.csv  ./intermediate_files/genus_species.tsv ./results/knn_result_k_11_structure_genus.csv
./src/run_knn.sh ./intermediate_files/structure_knn_df.csv  ./intermediate_files/order_species.tsv ./results/knn_result_k_11_structure_order.csv
./src/run_knn.sh ./intermediate_files/structure_knn_df.csv  ./intermediate_files/class_species.tsv ./results/knn_result_k_11_structure_class.csv
./src/run_knn.sh ./intermediate_files/structure_knn_df.csv  .intermediate_files/phyla_species.tsv ./results/knn_result_k_11_structure_phyla.csv
```

### Running the Robustness Analysis
To figure out if the classification framework was robust to feature removal the following script was run with removal percentages.
Each iteration will run within a new directory in the results folder
```
./src/run_robustness_analysis.sh  ../../intermediate_files/structure_knn_df.csv ../../intermediate_files/family_species.tsv 
```

### Running the Human-specific KNN classification
Intermediate files used are:
+ `intermediate_files/Uniprot_taxonomy_host_9606_2025_07_14.csv`- Uniprot labels of human-infecting virus
+ `intermediate_files/human_infecting_viruses_ncbi.csv` - human infecting viruses, extracted from Host column of `ncbi_viruses_genome_lengths.csv`
+ `intermediate_files/species_taxid.tsv` - a mapping of species name to taxid for human tropism matching

The following arguments are (1) outpath, (2) input, (3) family to species mapping.
```
./src/run_human_knn.sh ./results/ ./intermediate_files/structure_df.csv ./intermediate_files/family_species.tsv
```

### Looking for Prevalent Human-Specific Features Across Viral Families

Intermediate files used are 
+ `intermediate_files/Uniprot_taxonomy_host_9606_2025_07_14.csv`- Uniprot labels of human-infecting virus
+ `intermediate_files/human_infecting_viruses_ncbi.csv` - human infecting viruses, extracted from Host column of `ncbi_viruses_genome_lengths.csv`

```
# Can be run with
 python3 ./src/human_spec.py ./intermediate_files/nomburg_files/merged_clusters.tax.tsv
```
The result in Fig4C is generated as `human_specific_prots_neg_set.png`. 

### Running the LTSM to identify mutated regions in the *Adenoviridae* vs *Parvoviridae*
Intermediate files used are:
+ `intermediate_files/aligned-psi-blast-relative-all.fa`- multiple sequence alignment of the PSI blast results of Parvoviridae Rep40
+ `intermediate_files/adeno.fa` - extracted Adenovirus hits from the above MSA
+ `intermediate_files/parvo.fa` - extracted Parvovirus hits from the above MSA

```
python3 ./src/model_regions.py 
```
Outputs are 1) graph with the sequence windows and corresponding AUROC, 2) csv with the beginning of the window coordinate and corresponding AUROC
